Игра - "Zombie-shoot"

описание идеи:
Игрок возраждается в определенном месте. Он бегает по карте и убивает зомби.
Зомби может напасть на человека и тогда вылетит скример (страшная картинка с резким звуком)
В зомби можно стрелять, чтобы он пропал.

подробное описание идей и их реализации:
на карте отмечена точка возрождения игрока, там он и возраждается.
в игре можно передвигаться по картеп используя кнопки WASD.
при нажатии ЛКМ происходит выстрел в точку курсора.
если зомби сопрекасается с игроком вылетает скример, который может напугать игрока.

использованы технологии:
1. pygame - для создания игры в целом
2. random - для разнообразия работы некоторых механизмов
3. прочие библиотеки для создания различный функций
